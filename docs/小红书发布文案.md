# 小红书发布文案

## 版本一：产品分享型
我把自己长期在用的一个 Mac 电池管理小工具开源了，名字叫：

`电池接管（Battery Takeover）`

它解决的是一个很具体的问题：
我长期插电办公，但不想让 MacBook 一直充到 100% 然后挂着。

所以我自己做了一个小发明，现在它已经能做到：
- 每分钟记录一次电池状态
- `92%` 停充，`88%` 恢复
- 开机自动运行
- 桌面点击进入
- 面板里直接看曲线、改阈值、查最近动作

我比较在意的一点是，它不是那种“只给你一个开关”的工具。

它会把每一次采样和动作都记下来，失败了也会自动降级，不会闷头继续写。所以对我来说，它更像一个“可解释的电池管理工具”，而不是玄学护电池。

仓库我已经公开了：
`https://github.com/yishu-ziyu/battery-takeover`

如果你也是长期插电办公党，欢迎试试。
