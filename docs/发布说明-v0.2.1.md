# v0.2.1 - 电池接管（Battery Takeover）

本次版本聚焦于控充行为的一致性与说明文档的收敛。

## 变更

- Dashboard 增加“项目电池管理”开关。
- 保存设置后立即应用当前模式与阈值，而不是只更新配置文件。
- 关闭项目管理时，主动清除已有充电限制，将充电行为交还给系统与底层后端。

## 影响

- 长期插电场景下，可以继续使用项目阈值控充。
- 临时出门场景下，可以直接关闭项目管理，恢复正常充电到 `100%`。
- 阈值修改后的行为与界面显示保持一致，不再依赖旧状态残留。

## 边界

- 本项目仍不提供硬件级“旁路电池”能力。
- 写入能力仍依赖第三方后端，当前优先兼容 `batt`。
- 当前验证环境仍以 `macOS 15.x + Apple Silicon` 为主。
- 安装包仍未做开发者签名与 notarization。
