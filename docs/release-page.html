边界说明：本项目不宣称硬件级“旁路电池”能力。当前实现是在 macOS 与后端工具允许范围内停止继续充电并维持区间；后端不可用或环境不满足时，会退回只读监控。

核心特性

把控充、记录和复盘做成一个本地闭环，不上传数据，也不依赖云端服务。

阈值控充

长期插电时按 92/88 一类策略维持目标区间，减少长时间满充。

本地记录

样本、动作、日报都保存在本机，方便复盘最近 24 小时曲线和策略执行。

安全降级

当写入后端不可用、执行失败或环境不满足要求时，自动退回只读监控。
