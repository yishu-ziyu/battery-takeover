# v0.2.0 - 电池接管（Battery Takeover）Installer Release

这一版的重点不是功能点，而是分发能力。

## 新增
- `macOS .pkg` 一键安装包
- 安装后自动完成：
  - 运行副本落地
  - LaunchAgent 安装
  - 桌面入口创建
  - 首次打开产品界面
- GitHub Actions 自动构建安装包
- GitHub Release 可直接挂载安装资产

## 对用户意味着什么
现在不需要再：
- 手动创建虚拟环境
- 手动执行安装脚本
- 手动配置桌面入口
- 手动处理开机自动运行

用户只需要从 GitHub Releases 下载 `.pkg`，双击安装，就能得到和演示一致的产品体验。

## 仍然保留的边界
- 当前首发仍以 `macOS 15.x + Apple Silicon` 为主要验证环境
- 控充能力仍依赖第三方后端，优先 `batt`
- 未做开发者签名与 notarization，首次安装可能仍会遇到 macOS 的安全提示
