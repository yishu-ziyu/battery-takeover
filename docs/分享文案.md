# 电池接管（Battery Takeover）分享文案

## 长文版
我把这几天做的一个 macOS 电池管理项目开源了，名字叫 `电池接管（Battery Takeover）`。

它解决的不是“玄学护电池”，而是一个很具体的问题：长期插电办公时，很多人不想让 MacBook 长时间停在 100%，但系统原生给的控制能力又不够细。

这个项目当前已经跑通了一个完整 MVP：
- 分钟级采集电池状态
- 按 `92%` 停充、`88%` 恢复做阈值控充
- 自动记录动作和样本
- 出现异常自动降级为只读
- 带一个轻量 Dashboard 和桌面点击入口

我比较在意的一点是，它不夸大能力。它并不宣称“物理绕过电池”，而是在 macOS 和现有后端允许的边界里，尽量把控充、记录和复盘这件事做扎实。

仓库地址：
`https://github.com/yishu-ziyu/battery-takeover`

欢迎试用、提 issue，也欢迎一起把这个项目做得更稳。
