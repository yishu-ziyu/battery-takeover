# v0.1.0 - 电池接管（Battery Takeover）MVP

这是 `电池接管` 的第一个公开版本，目标是把一个可真实运行的 macOS 电池管理 MVP 分享出来。

## 本版包含
- 分钟级电池样本采集
- `92%` 停充 / `88%` 恢复的阈值控充
- `OBSERVE_ONLY / ACTIVE_CONTROL / DEGRADED_READONLY` 三态运行模式
- SQLite 审计与复盘数据落库
- 日报 Markdown 生成
- LaunchAgent 开机自动运行
- 轻量 Dashboard
- 可点击桌面 App 入口

## 关键设计原则
- 不虚构能力：不承诺物理层面的“适配器完全绕过电池”
- 真实可跑：优先交付本机可运行闭环，而不是只做原型界面
- 失败可退：执行异常自动降级为只读监控
- 过程可查：样本、动作和状态均可回放

## 已知边界
- 当前主要在 `macOS 15.x + Apple Silicon` 环境验证
- 控充依赖第三方后端，当前首选 `batt`
- 个别系统命令在边缘场景下可能变慢，后续还会继续优化容错
